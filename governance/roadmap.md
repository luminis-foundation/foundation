# Luminis Foundation — Roadmap

Forward-looking priorities. **Draft — to be ratified by the Board.**

## Near term
- Obtain IRS §501(c)(3) determination letter
- Minimum viable sensor node: single piñon root zone, ESP32 + ADS1115/ADS1299, sensing-only mode
- Establish baseline mycelial spike and EI/EC profiles across a full seasonal cycle (Pecos watershed)

## Mid term
- Site-specific quantized TinyML classifier for mycorrhizal health state
- First public dataset release
- Formal specification of the federated model-update protocol (node weighting by mycorrhizal health score)

## Long term
- Inter-watershed federation trials
- Governance tooling (biological threshold-gating at grove scale; optional cryptographic coordination at inter-watershed scale)
